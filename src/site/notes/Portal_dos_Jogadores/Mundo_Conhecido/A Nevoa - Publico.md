# A Névoa

A névoa apareceu para cada personagem de uma forma diferente.

Para alguns, ela veio como perseguição. Para outros, como sentença. Para outros, como consequência de um erro arcano.

## O que vocês sabem

- Ela não parece natural.
- Ela alcançou pessoas em lugares diferentes.
- Ela conduziu todos para a mesma estrada escura.
- Ela parece ligada à entrada em [[Portal_dos_Jogadores/Mundo_Conhecido/Barovia - Publico\|Baróvia]].

> [!quote] Sensação
> A névoa não parece só esconder o caminho. Ela parece escolher quem passa.
